## Declarations

Variable declaration is an important part in any programming language. 

<!--DOCUSAURUS_CODE_TABS-->
<!--JavaScript-->
```js
// const
const msg = "Hello world";
const num = [1,2,3,4,5];

// var
var d = new Date();

// let
let a;

// const
const msg = "Hello world";
const num = [1,2,3,4,5];
```

<!--Python-->
```py
# int
one = 1
two = 2

# decimal
dem = 1.1

# string
strval = "Hello"
```

<!--Swift-->
```swift
// 1
var/let name : type = value

// 2
var/let name : type
name = value

// 3
var/let name = value
```
